# Color Converter

Free online tool to convert between HEX, RGB, HSL color formats.

## Features

- ✅ HEX to RGB conversion
- ✅ RGB to HSL conversion
- ✅ Live color preview
- ✅ One-click copy for all formats
- ✅ CSS code generator
- ✅ Color picker
- ✅ Mobile responsive

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- CSS Modules

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
# Output in /dist folder
```
